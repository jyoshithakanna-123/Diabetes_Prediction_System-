# 🏥 Diabetes Prediction System with Interactive Dashboard

An end-to-end Machine Learning project built to predict diabetes risk based on medical metrics.

## 🚀 Features
* *Exploratory Data Analysis:* Statistical summaries and class balance profiling using Pandas.
* *Data Visualizations:* Dynamic correlation heatmaps, feature distribution boxplots, and trend checks using Seaborn.
* *Machine Learning Model:* Standardized clinical feature distributions using StandardScaler and a trained linear Support Vector Machine (SVM) classifier.
* *Interactive UI:* A real-time diagnostic slider application designed using ipywidgets for interactive health risk assessments.

## 🛠️ Tech Stack & Dependencies
* Python 3 & Google Colab
* NumPy & Pandas (Data Wrangling)
* Matplotlib & Seaborn (Visualizations)
* Scikit-Learn (Machine Learning Core)
* Ipywidgets (Frontend Interface Components)
